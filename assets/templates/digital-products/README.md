# Digital Product Landing Templates

Standalone HTML Mode landing page templates for separate digital products. Each file has product-specific copy, SEO metadata, and a distinct visual theme.

## Files

| Product | File | Theme |
|---|---|---|
| Template Keuangan | `template-keuangan.html` | Calm ledger / green finance |
| Template Keuangan Bisnis | `template-keuangan-bisnis.html` | Dark business console |
| Template Keuangan Rumah Tangga | `template-keuangan-rumah-tangga.html` | Warm home budget |
| Template Keuangan Mahasiswa | `template-keuangan-mahasiswa.html` | Campus notebook |
| Ebook Financial | `ebook-financial.html` | Editorial finance |
| Ebook Productivity | `ebook-productivity.html` | Bento productivity |
| Ebook AI | `ebook-ai.html` | Neon AI lab |

## Before Importing to Scalev

- Replace brand/footer text.
- Replace final CTA `href="#checkout"` with the real checkout URL or section target.
- Adjust price, offer, bonus, and refund/access policy copy.
- Replace placeholder proof/testimonial blocks with real proof when available.
- Keep financial disclaimers unless a qualified reviewer approves different wording.

## Design Notes

- Mobile-first layout with inline CSS.
- Product-specific SEO title, meta description, Open Graph, Twitter card, and `robots="index, follow"`.
- No external JS frameworks.
- Google Fonts only.
- No analytics or tracking pixels embedded.
